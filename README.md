## Research

**Master's Thesis @ Fraunhofer FOKUS** *(2024 – 2025)*  
*A STUDY OF REAL-WORLD STRUCTURAL KNOWLEDGE BASE INTEGRATION METHODS IN KBLAM AND THEIR IMPACT ON LLM RESPONSE PERFORMANCE*

Fine-tuning a LLaMA-based model using a KBLAM-inspired architecture that injects structured knowledge entries directly into transformer attention layers — enabling improved open-ended QA without retrieval overhead.

`PyTorch` · `Transformers` · `Knowledge Graphs` · `HPC / Distributed Training`
